# WALKER PRO DJ - Deployment Guide

## Cara Deploy ke GitHub Pages (Web)

Aplikasi ini sekarang dikonfigurasi untuk berjalan sebagai Website di GitHub Pages.

1. **Push perubahan** ke GitHub (Klik tombol **Deploy** atau **Sync** di AI Studio).
2. Buka repositori Anda di GitHub: `https://github.com/kamudanaku652-del/DJ-app`
3. **PENTING:** Klik tab **"Settings"** di atas.
4. Di sidebar kiri, klik **"Pages"**.
5. Di bagian **"Build and deployment"**, cari menu **"Source"** dan ubah menjadi **"GitHub Actions"**.
6. Klik tab **"Actions"** di atas untuk melihat workflow **"Deploy to GitHub Pages"** berjalan.
7. Setelah centang hijau, website Anda akan tersedia di link yang muncul (biasanya `https://kamudanaku652-del.github.io/DJ-app/`).
